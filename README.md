# Glaze-RL

Glaze-RL is a multi-purpose library designed to evaluate and improve the level of "glazing" across LLM's with the following features:
- Synthetic Data Generation
- LLM "Glazing" Evaluation
- Reinforcment Learning Environment for LLM's

With AI Induced Psychosis on the rise, it's important to be able to evaluate and improve an LLM's ability to push back on illogical user prompts.
